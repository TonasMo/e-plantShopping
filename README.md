# coding-project-template

Neste projeto final com o nome de e-plantShopping, criou-se um aplicativo de carrinho de compras para uma loja online de plantas que oferece uma variedade de plantas de interior.
As funcionalidades do carrinho de compras Paradise Nursery incluiem:
Uma página inicial com um botão que liga à página de listagem de produtos
Uma barra de navegação com links para as páginas inicial, listagem de produtos e carrinho de compras
Um cartão para cada planta que exibe as diferentes plantas juntamente com suas imagens, nome, descrição, custo e um botão Adicionar ao carrinho.
Tem duas seções descrevendo as plantas naquela seção. Por exemplo, “Plantas Aromáticas” e “Plantas Medicinais”.
Uma página de carrinho que exibe os produtos no carrinho.
O carrinho tem um cartão para cada tipo de planta no carrinho. Cada cartão  tem a miniatura, o custo unitário, o custo de todas as plantas daquele tipo e botões para aumentar e diminuir a quantidade, juntamente com um botão Excluir.
Botões Continuar Comprando e Finalizar Compra
